# Screenshot Checklist

Capture these four images before sending the portfolio link:

1. `dashboard-desktop.png`: Admin or Manager dashboard at desktop width.
2. `mobile-dashboard.png`: Mobile dashboard with drawer or notification panel visible.
3. `deals-pipeline.png`: Pipeline board showing multiple deal stages.
4. `customer-profile.png`: Customer profile with activity timeline.

Recommended browser sizes:

- Desktop: 1440 x 900
- Mobile: 390 x 844
